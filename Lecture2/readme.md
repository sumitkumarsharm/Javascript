### Arithmetic Operators

Arithmetic operators in JavaScript allow us to perform mathematical calculations such as addition, subtraction, multiplication, division, and modulus. These operators are important for performing basic mathematical calculations within our code.

1. Addition

```
let firstNumber = 25;
let SecondNumber = 254;
let result = firstNumber + SecondNumber;
console.log(result); // Output: 279
```

2. Subtraction

```
let firstNumber = 25;
let SecondNumber = 254;
let result = firstNumber - SecondNumber;
console.log(result); // Output: -229
```

3. Multiplication

```
let firstNumber = 25;
let SecondNumber = 254;
let result = firstNumber * SecondNumber;
console.log(result); // Output: 6350
```

4. Division

```
let firstNumber = 25;
let SecondNumber = 254;
let result = firstNumber / SecondNumber;
console.log(result); // Output: 0.09803921568627451
```

5. Modulus

```
let firstNumber = 25;
let SecondNumber = 254;
let result = firstNumber % SecondNumber;
console.log(result); // Output: 25
```

6. Exponentiation

```
let firstNumber = 25;
let SecondNumber = 254;
let result = Math.pow(firstNumber, SecondNumber);
console.log(result); // Output: 1.4190625e+66
```

7. Increment

```
let firstNumber = 25;
let result = firstNumber++;
console.log(result); // Output: 25
console.log(firstNumber); // Output: 26
```

8. Decrement

```
let firstNumber = 25;
let result = firstNumber--;
console.log(result); // Output: 25
console.log(firstNumber); // Output: 24
`
```
